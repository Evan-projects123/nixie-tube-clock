# Nixie Tube Clock

A retro-style Nixie tube clock simulation with multiple functions:
- Clock display with battery monitoring
- Timer
- Stopwatch
- Metronome
- Alarm with customizable sounds

## Features
- Authentic Nixie tube glow effects
- Battery status monitoring
- Procedural sound generation
- Multiple display modes
- Marquee text display
- Touch and mouse controls

## Usage
- Click on clock panel to cycle through modes
- Click on marquee panel to enter settings
- Right-click/context menu for additional controls
- Keyboard arrows and mouse wheel for adjustments

## Installation
1. Clone the repository
2. Open `index.html` in a web browser
3. No build process required

## Font
The project uses a Nixie-style font. You can replace it with any font of your choice.
